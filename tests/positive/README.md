## Project Description
This project contains automated end-to-end tests written using Playwright
to test a Singlish to Sinhala translation web application. The test suite
includes positive, negative, and UI test cases based on the provided test
case specification.

## Technologies Used
- Node.js
- Playwright
- TypeScript

## How to Install Dependencies
Make sure Node.js is installed.

## How to Run All Tests
npx playwright test

## How to View HTML Test Report
npx playwright show-report
